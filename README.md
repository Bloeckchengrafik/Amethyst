# Amethyst
An electron-based audio player with a pixel-art aesthetic

```powershell
winget install amethyst
```

<br>
<br>

## Contributing
- Use `Node.js 1.16`
- Use the `recommended extensions`
- Have `GCC` installed

## Features
- Custom cursors
- Animated GIF cover arts
- Quality-based scoring system
